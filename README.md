# Student CRUD Application

A full-stack CRUD application built using Spring Boot, JPA, MySQL, and Thymeleaf.

## Features
- Create, Read, Update, Delete students
- MVC architecture
- Database integration using Spring Data JPA
- Thymeleaf-based UI
- Post-Redirect-Get pattern

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf

## What I Learned
- Entity management with JPA
- Handling common Spring Boot errors
- Thymeleaf data binding
- Controller–Service–Repository flow
